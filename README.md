# PROJETO AVALIATIVO 02 - PROGRAMAÇÃO DE SÍTIOS

## 🎯 Objetivo do Projeto

Criar uma interface web funcional, responsiva e com boa experiência de uso, que consuma os endpoints do backend e permita ao usuário realizar as operações previstas no contrato da API.

---

## 💻 Tecnologias Utilizadas

### Linguagens Utilizadas
- HTML
- CSS
- VANILHA JS
- BOOTSTRAP: CSS para o layout responsivo

### APIs Utilizadas
- RENDER: responsável pelo CRUD
- Cloudinary: Converter imagem em link

## Acesso das APIs
- Render: https://render.com/
- Cloudinary: https://cloudinary.com/
---
## 📌 Funções implementadas 

### CRUD
- GET: Listagem
- POST: Publicação/Cadastro
- PUT: Alteração
- DELETE: Exclusão

As páginas foram criadas em Javascript e renderizadas em tempo real no html.

---

## 🔎 Estrutura do Projeto

```
P2-PROGRAMACAO-SITIOS/
├── css/
│   └── style.css           -- Estilo da página
├── img/
├── js/
│   ├── functions/
│   │   ├── auth.js         -- Autenticação para usar o render
│   │   ├── cloudinary.js   -- Converte imagem em link
│   │   ├── preview.js      -- Gerar preview da imagem quando é upada
│   │   └── produtos.js     -- Back End do crud
│   └── pages/              -- Páginas feitas em Javascript
│       ├── cadastro.js 
│       ├── consulta.js
│       └── editar.js
├── app.js                  -- Responsável pela renderização das páginas
└── index.html              -- Local onde as páginas vão ser renderizadas
```
